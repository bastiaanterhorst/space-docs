## Global hotkeys

In addition to the predefined shortcuts below, you can configure two custom global hotkeys that will also work
when Space is in the background. You can use these hotkeys to add a new task to your inbow, or to today. 
Configure these hotkeys in Settings &rarr; Tasks.

## General

| Key                           | Description                     |
|-------------------------------|---------------------------------|
| <kbd>Cmd</kbd> + <kbd>N</kbd> | New task (on the current screen) |
| <kbd>Cmd</kbd> + <kbd>\\</kbd> | Jump to any screen in the app |
| <kbd>Cmd</kbd> + <kbd>1</kbd> | Jump to inbox |
| <kbd>Cmd</kbd> + <kbd>2</kbd> | Jump to day |
| <kbd>Cmd</kbd> + <kbd>3</kbd> | Jump to week |
| <kbd>&darr;</kbd> / <kbd>&uarr;</kbd> | Navigate task lists |
| <kbd>Alt</kbd> + <kbd>&darr;</kbd> / <kbd>&uarr;</kbd> | Jump to bottom / top of task list |
| <kbd>Shift</kbd> + <kbd>&darr;</kbd> / <kbd>&uarr;</kbd> | Select downward / upward in task list |
| <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>&darr;</kbd> / <kbd>&uarr;</kbd> | Select all downward / upward in task list |
| <kbd>&larr;</kbd> / <kbd>&rarr;</kbd> | Navigate weekdays on the week screen |

## On day or week screens

| Key                           | Description                     |
|-------------------------------|---------------------------------|
| <kbd>Cmd</kbd> + <kbd>B</kbd> | Plan the current day or week |
| <kbd>Cmd</kbd> + <kbd>&larr;</kbd> | Go back one day or week |
| <kbd>Cmd</kbd> + <kbd>&rarr;</kbd> | Go forward one day or week |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>&larr;</kbd> | Go to the current day or week |


## With selected tasks

| Key                           | Description                     |
|-------------------------------|---------------------------------|
| <kbd>Enter</kbd> | Toggle editing the task |
| <kbd>Cmd</kbd> + <kbd>.</kbd> | Open task in focus view (full screen editing) |
| <kbd>Cmd</kbd> + <kbd>G</kbd> | Open task in floating panel (handy for tasks on week screen) |
| <kbd>Cmd</kbd> + <kbd>D</kbd> | Move tasks |
| <kbd>Cmd</kbd> + <kbd>S</kbd> | Schedule tasks |
| <kbd>Cmd</kbd> + <kbd>Y</kbd> | Quick schedule for today |
| <kbd>Cmd</kbd> + <kbd>U</kbd> | Quick schedule for tomorrow |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>U</kbd> | Quick schedule for next monday |
| <kbd>Cmd</kbd> + <kbd>I</kbd> | Quick schedule for this week |
| <kbd>Cmd</kbd> + <kbd>O</kbd> | Quick schedule for next week |
| <kbd>Cmd</kbd> + <kbd>P</kbd> | Quick unschedule |
| <kbd>Cmd</kbd> + <kbd>E</kbd> | Toggle completion |
| <kbd>Cmd</kbd> + <kbd>R</kbd> | Toggle in-progress |
| <kbd>Cmd</kbd> + <kbd>T</kbd> | Toggle priority |
| <kbd>Cmd</kbd> + <kbd>K</kbd> | Go to scheduling of task (day or week) |
| <kbd>Cmd</kbd> + <kbd>L</kbd> | Go to categorisation of task (project or area)  |

## Hovering over calendar event

| Key                           | Description                     |
|-------------------------------|---------------------------------|
| <kbd>Cmd</kbd> + <kbd>J</kbd> | Join video call (if attached to the event) |
